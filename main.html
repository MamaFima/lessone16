Основные теги для работы с таблицами на странице. Тег
используется для представление таблиц в HTML. Для создания строки таблицы прописывается тег (это один горизонтальный ряд). Тег определяет ячейку таблицы которая содержит данные. В ячейку можно прописывать и изображение, и видео, и аудио контент. Если необходимо прописать заголовок применяется тег . Для придания стиля полученной таблице можно воспользоваться атрибутами: атрибут bgcolorуказывает цвет фона ячейки заголовка таблицы; атрибут align задает горизонтальное выравнивание содержимого абзаца; атрибут border позволяет установить рамки (толщину) вокруг элемента; атрибут height устанавливает высоту таблицы; атрибут width устанавливает ширину таблицы.
